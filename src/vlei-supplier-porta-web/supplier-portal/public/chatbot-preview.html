  1. Run the development server: npm run dev
  2. Navigate to /company route

  4. Click the purple chatbot button in the bottom-right corner
  5. Start chatting with the AI assistant!

Try these commands:

  • help - See all available features
  • create order - Get order creation assistance
  • check status - View order filtering options
  • supplier - Learn about supplier management
